Requisitos

A seleção do operador deve preencher o tipo de jogo (operator game type).
A seleção do tipo de jogo deve preencher o nome do slate (operator name).
A seleção de um tipo de jogo deve mostrar uma lista de jogadores na parte inferior com o primeiro jogador selecionado.
O usuário deve poder clicar em qualquer um dos jogadores e o componente do jogador deve ser atualizado de acordo.
A paginação deve funcionar.
A página deve ser responsiva para layouts de desktop (md e superiores).
A responsividade móvel não é um requisito.

Entregáveis

Arquivo completo do código-fonte utilizando React Vite.
Deve usar TypeScript.
Deve usar Tailwind CSS. Para os selects (combobox) deve se utilizar o Tailwind Forms.
Para ícones deve se utilizar o Lucide Icons.
